## Configurações
Todo email enviado no ambiente de desenvolvimento é capturado e enviado para um único destinatário.
Para isso, adicione no seu `.profile`, `.bashrc` ou `.zshrc`:

    export GMAIL_USR="email usado na config smtp"
    export GMAIL_PWD="senha do email na config smtp"
    export DEV_MAIL_TO="email que receberá as mensagens"
